# 🐱 KASP-GOTCHI — Tamagotchi on Kasplex L2

Raise, evolve, and battle digital pets using the Kaspa blockchain.

## 📦 Tech Stack
- Solidity (ERC721)
- Hardhat + Chai
- Next.js + TailwindCSS + Framer Motion
- wagmi + RainbowKit
- Kasplex L2 Testnet

## 🚀 Getting Started

```bash
# Install dependencies
yarn install

# Compile and test contracts
yarn hardhat compile
yarn hardhat test

# Run frontend
cd frontend
yarn dev
